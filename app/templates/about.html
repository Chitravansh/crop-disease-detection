About ArogyaFasal

ArogyaFasal is an AI-powered crop disease detection system designed to help farmers and agricultural researchers identify plant diseases quickly using leaf images. The system uses deep learning to analyze leaf patterns and detect possible infections.

How the System Works

Upload Image

User uploads a leaf image using the web interface.

AI Analysis

The CNN model analyzes visual patterns in the leaf.

Disease Detection

The model predicts crop type and disease class.

Key Features

  • Detects diseases across 38 crop classes
  • AI-based image classification using CNN
  • Drag-and-drop leaf image upload
  • Real-time prediction with confidence score
  • Interactive web interface using Flask
